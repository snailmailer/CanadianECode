# Your Guide to the Canadian Electrical Code (2024 26th Edition)

Welcome to the interactive guide for the 2024 Canadian Electrical Code! This application provides a modern, fast, and accessible way to navigate and read critical sections of the CEC Code book.

## Features

- **Quick Navigation**: Use the sidebar to instantly jump between sections, appendices, tables, and diagrams.
- **Clean Interface**: The viewer provides an distraction-free reading experience, formatting the code into easily digestible paragraphs.
- **Text-to-Speech (TTS)**: Don't want to read? Click the "Read Aloud" button at the top of any section to have the application read the electrical code to you using native browser speech synthesis.

## How to Use the Website

### Navigating the Guide
1. **Sidebar Menu**: On the left side of the screen, you will find a list of all available Sections, Appendices, and Tables.
2. **Select a Topic**: Click on any item in the sidebar (e.g., "Section 10 — Grounding and bonding"). The content will immediately load in the main viewing area on the right.
3. **Scroll to Read**: You can scroll through the content just like a normal webpage.

### Using the Text-to-Speech Feature
1. **Start Reading**: Once a section is open, look for the **"Read Aloud"** button at the top right of the content header. Click it to start the audio playback.
2. **Stop Reading**: The button will turn green and say **"Stop Reading"** while audio is playing. Click it again to stop the playback at any time.
3. *Note: If you click on a new section in the sidebar while the audio is playing, the playback will automatically stop.*

## Local Development Setup

If you have downloaded this source code and wish to run it locally on your machine, follow these steps:

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your computer.

### Installation
1. Open a terminal/command prompt.
2. Navigate to the project directory:
   ```bash
   cd path/to/CanadianECode
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the local development server:
   ```bash
   npm run dev
   ```
5. Open your web browser and go to the URL provided in the terminal (usually `http://localhost:5173`).
